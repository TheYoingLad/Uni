Számrend

Parancsok

  • ls fájlok listázása az aktuális könyvtárban
    • ls some_folder some_folder fáljainak listázása
    • ls -a rejtett elemek megjelenítése
    • ls -l több info megjelenítése
    • ls -la előző két opció kombinációj
  • pwd aktuális munkakönyvtár elérési útja
  • cd könyvtár váltás
  • mkdir könyvtár létrehozása
  • rmdir könyvtár eltávolítása
  • cp fájl / könyvtár másolása
  • mv fájl / könyvtár áthelyezése
  • rm fájl / könyvtár törlése
  • cat (általában szöveges) fájlok képernyőre írása, konkatanációja
  • clear karakteres képernyő törlése
  • ssh távoli számítógép elérése (többek között)
